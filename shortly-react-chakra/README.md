# Shortly Website

## Usage :

<hr>

> ## `npm install`
>
> Installs the required dependencies

> ## `npm start`
>
> Runs the app in the development mode.<br>
> Open [http://localhost:3000](http://localhost:3000)

> ## `npm run build`
>
> Builds the app for production to the `build` folder.<br>
