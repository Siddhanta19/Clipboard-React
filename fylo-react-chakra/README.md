# Fylo Website

## Usage :

<hr>

> ## `yarn install`
>
> Installs the required dependencies

> ## `yarn start`
>
> Runs the app in the development mode.<br>
> Open [http://localhost:3000](http://localhost:3000)

> ## `yarn build`
>
> Builds the app for production to the `build` folder.<br>
